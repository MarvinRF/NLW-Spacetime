## 🖥️ Projeto
Este é um Projeto Web Responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Este é um projeto desenvolvido durante o evento NLW da Rocketseat com as seguintes Tecnologias:

- HTML5;
- CSS3 (Flexbox e Grid);
- Git (GitHub).
